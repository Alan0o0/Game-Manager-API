# Game Manager API

Projeto backend para gerenciamento de jogos, permitindo criar, listar, buscar por ID e mover jogos entre listas.

---

## Sobre o projeto

API desenvolvida com arquitetura em camadas, separando responsabilidades em Domain, Repository, Service e Controller.  
Utiliza ORM e possui seed automática do banco de dados.

O desenvolvimento ocorreu entre 19 e 23 de maio de 2025.

---

## Funcionalidades

- Criar jogos  
- Listar jogos  
- Buscar jogo por ID  
- Obter listas  
- Mover jogos entre listas  
- Seed do banco de dados
